Project: Integration of AION VM into the RIF Consensus Node

The goal of this project is to:

Include the popular language Java as an alternative for coding smart contracts, the AVM (AION Virtual Machine) is a runtime environment just on top of the JVM that allows developers to run Java bytecode on the blockchain. The biggest challenge is to converge different technologies in the same project where each one has its own particular characteristics.

The impact of this project will be to facilitate the adoption of blockchain technology for those companies interested in blockchain, avoiding the cost and time necessary to train their teams in unfamiliar frameworks, languages and tools.

Miscellaneous: the project is a great challenge and needs a lot of detail to make all the pieces fit seamlessly.
