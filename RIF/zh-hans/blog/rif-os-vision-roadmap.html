RIF OS 愿景与路线图

撰文：Gabriel Kurman——RIF 战略顾问
2019年7月4日

比特币与区块链技术引发了价值互联网的革命。点对点网络、加密、主权身份和开源软件正改变着用于统治世界的典范。不过，相比集中式解决方案，我们仍无法为开发者和数字用户提供无缝的用户体验。如我们希望区块链技术让世界享受到去中心化、无需许可互联网的益处，我们需要使其变得更易于使用、更快和可以扩展。

RIF OS 愿景

RIF OS（RSK 基础架构框架开放标准）旨在促进开发者和用户与区块链技术的交互，并实现大规模采用。 RIF OS 是一套开放和去中心化的基础设施协议，可在统一环境中实现分布式应用程序 (dApp) 更快、更轻松和可扩展的开发。

通过 RSK 网络开发的 RIF 服务将解决此生态系统中最重要的门限，以促进开发安全且方便用户使用的分布式应用程序 (dApp)：

  • RIF 名称服务 (RNS) 和 RIF 目录： RIF 名称服务旨在通过一种架构来为用户提供更友好的体验，这种架构可以提供人类能够理解的名称或别名，来识别区块链的地址。随着通过添加名称解析服务或“别名”实现“易用性”，出错概率可大幅降低。
    RNS 是 RIF 目录框架的组成部分，该框架还将包括动态地址解析（握手）、支付渠道 ID、去中心化存储 ID、社交网络句柄，去中心化 ID 和去中心化加密信誉。 (https://www.rifos.org/nameservice.
  • RIF Lumino 网络和 RIF 支付： Lumino 是一个链外支付渠道网络，可用于 RSK 网络上的每个代币，以极低成本使每秒交易次数达到上千次。RIF Lumino 是 RIF 支付框架的一部分，还可实现与 Lightning 网络和其他链外支付网络（包括跨区块链和跨代币交易）的互通性。RIF Lumino 网络具备近乎即时的处理能力，网络交易成本不足 1 美分，其提供必要基础设施，以为建设未来的金融体系奠定坚实基础。RIF Lumino 网络的推出是迈向可扩展网络基础设施的第一步。(https://www.rifos.org/rif-lumino-network/).
  • RIF 存储： RIF 存储是一个统一的界面以及库集合，用于促进加密和去中心化的存储并简化信息。RIF 存储旨在允许用户以安全的方式存储和共享所有敏感数据。RIF 存储将针对不同需求提供多个选项，从去中心化 Swarm 存储到加密云和物理存储。
  • RIF 通信： 中心化社交网络、电信公司和科技巨头正开始创新，并冒险进入支付和价值交换中。不过，他们通过其数十年来一直使用的相同的集中化、审查和数据收集标准来做这件事。RIF 通信旨在实现用于中继区块、支付和代币的点对点去中心化网络，还能够传输加密消息、聊天记录和信息。RIF 通信旨在成为一组 API，允许各方注册其通信方法，发现其他方，并通过安全的方式联系它们，以确保信息的机密、完整和真实性。
  • RIF 数据馈送： 许多复杂的智能合约解决方案需要访问真实的数据馈送 (oracles)。RIF 数据馈送旨在成为一项协议，允许去中心化应用程序以安全、防篡改和信任最小化的方式访问信息，保证从外部数据源和提供者获取的值的确定性输出（所有矿工节点必须获得特定查询请求的相同值）。 RIF 数据馈送还将包括一个“拉模式”选项和“订阅模式”，消费者可通过“拉模式”根据每次查询支付和检索数据，以及通过“订阅模式”支付固定价格以访问可由服务提供商缓存并定期更新的数据。

RIF OS 路线图

在 RIF OS 框架推出以来的前八个月内，我们已多次推出和发布 RIF 的不同服务。

  • 2018 年 11 月：RNS 1.0 推出，RNS简化了区块链 域名系统 (DNS)，方法是将看起来像一串随机字母和数字的传统区块链域转化成人类可以理解的形式，例如名称或容易记住的单词，这大大降低了复杂性并减少了错误，同时开放了区块链域的可访问性。
  • 2019 年 5 月：RIF Lumino 网络推出，这是比特币区块链的第三层解决方案，它为基于RSK构建的每种代币启用状态通道，从而提高了交易吞吐量，并将成本降低了几个数量级。
  • 2019 年 6 月：RNS 2.0 多区域链解析器推出，RIF 名称服务成为世界上唯一一个允许管理基于任何平台构建的代币的域名和别名的服务，增强了互通性，使用户更易于在不同网络上完美转移资产 。几次的发布允许用户将 RNS 集成到多个钱包中，并用其来解析 RSK、BTC 和任何其他区块链地址。

但这仅仅是一个开始，在接下来的 12 个月内，IOVLabs 团队将致力于通过 RIF OS 框架提供以下技术：

  • 2019 年第 3 季度
    • RNS 1.1 升级（通过直接购买域名进行拍卖取消和替换）
      • RIF Lumino：轻客户端 + 开发票 + 软件开发工具包 (SDK)
        • 轻客户端：使轻客户端能够连接到 RIF Lumino 网络，而无需运行完整节点。（移动集成所需）
        • 开发票：为 Lumino 支付添加开发票功能（用于请求付款的预定义二维码）
        • SDK：用于连接 RIF Lumino 节点的 JavaScript SDK（更易于进行移动集成所需）
      • RIF 存储：RSK 测试网上的测试版（这将允许以加密和去中心化的方式存储文件）和 RNS 集成（RIF 存储也将受益于 RSK 和其他 RIF 服务之间的多种协同作用）
  • Cuarto trimestre de 20192019 年第 4 季度
    • RNS 反向解析（允许搜索特定域名）和托管服务（相似 GoDaddy 允许无信任销售/购买 RNS 域名）
    • RIF 支付：中心费用（允许中心收取交易中继费用以获得营运资金。对提升网络可扩展性非常重要）
    • RIF 存储：RSK 主网发布加上 RIF Lumino 集成（Lumino允许为去中心化存储支付经济激励）
    • RIF 数据馈送：测试版 (MVP)
  • 2020 年第 1 季度
    • RNS：与 DNS 集成（允许互联网域名所有者在 RNS 上注册相同域名）
    • RIF 支付：Lightning 集成（通过 RIF 支付库）和跨代币支付（RIF Lumino 网络中代币之间的支付和原子交换）
    • RIF 存储：加密云存储（允许用户使用相同界面轻松加密其在云中的信息。非常适合无法进行集群存储的大型文件）
  • 2020 年第 2 季度
    • RNS 和 RIF 通信：RNS 动态解析（“握手”。允许使用相同域 ID 动态解析多个地址）
    • RIF 支付：瞭望塔（为用户提供安全保障，以防交易对方趁用户离线时拿走其资金。移动 dApp 的密钥）和跨区块链支付 （允许在 Lightning 开始付款并用 Lumino 代币完成付款，反之亦然）
    • RIF 存储：永久存储（允许用户向网络付费，以存储不受欢迎的内容，并且每次检索时付费的模型不起作用）以及完整的 UI 和库，以使用户界面更易于使用和集成

RIF OS 协同作用

除了单项 RIF 服务的创新之外，IOVlabs 开发的整个 RSK / RIF 完整堆栈中的协同作用还具有很大价值。该等协同作用来自共享基础架构、协调一致的路线图以及共同愿景——更好地服务可为数百万用户服务的用例。部分协同作用为:

  • 通过 RNS 实现 RSK 和 RIF 一般对等发现
  • 通过 RIF Lumino 实现 RSK 网络可扩展性
  • 通过 RIF Lumino 提供 RIF 存储激励
  • RIF 通信作为 RSK 和 RIF 的通用对等路由层
  • 面向 RSK 公共节点的 RIF 通信和 Lumino 服务
  • 用于 RNS/DNS 集成和 RSK 去中心化稳定币的 RIF 数据馈送

除了这些集成之外，我们还预计，通过让 RSK 联盟成员充当 RIF Lumino 中心和瞭望塔以及集成式 RSK/RIF 完整节点，可实现额外的基础设施协同作用。
