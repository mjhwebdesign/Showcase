RIF Name Service (RNS) & MyCrypto: RNS Manager 1.0.2 Now Supports MyCrypto

May 1, 2019

At IOVLabs, we are happy to announce that the new version of RNS Manager allows users to connect with MyCrypto for any operation such us registering and managing names.

MyCrypto, is one of the leading open-source, client-side tools for generating wallets, handling RRC-20 tokens, and interacting with the blockchain more easily. To register your domain in MyCrypto, simply check the domain status and follow the instructions. During the registration process and each name action, you will see the MyCrypto checking-out option to sign the RNS contracts interactions.

This new integration, shows one more time the close relationship between RNS and MyCrypto, the first integrated wallet supporting RIF Name Service to use ‘.rsk’ addresses!

The main objectives of RIF services is to facilitate the use of Blockchain technology by making it available for all. The RIF Name Service is particularly designed to make the user experience more friendly. This integration with MyCrypto, is a major milestone along this path.
