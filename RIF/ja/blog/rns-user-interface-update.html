RNSユーザーインターフェイスの更新

Ilan Olkies - RNS Product Owner
2020年4月8日

RIF 名称サービス（RNS）は、ユーザーが任意のブロックチェーンで読み取り可能なドメインを持つのを可能にする分散型サービスです。言い換えれば、RNS はブロックチェーンの電話帳です。人間は、alice.rskやbob.rskのようなドメイン名を介してオンライン上で情報にアクセスします。ウォレットは、アドレスを使用してブロックチェーンのプロトコルを通して対話をします。RNS はドメイン名をアドレスに翻訳するため、ユーザーが送金を行ったり Dapp やそのドメイン名を使用する他者と対話するのを可能にします。
例えば、あなたは RIF トークンを Alice に送金したいとします。あなたはまず、ウォレットを開き、Alice にアドレスを尋ねることから始めます。
0xfDB628524AD95c95a2C1f8dA9b8Bd92b6478CF6F

送金ボタンをクリックすると、ウォレットはあなたが入力したアドレスを受け取り、そのアドレスに送金するトランザクションにサインします。
ただし、それだけが送金方法ではありません！Alice が彼女のドメインを登録していれば、あなたのウォレット入力に彼女のドメインをタイプするだけで即座に送金することができます。
alice.rsk

ウォレットはドメインのアドレスを入手するために RNS を使用し、 RNSpublic phonebook 内で検索します。アドレス入手後、あなたは送金の受取人を使用してトランザクションにサインします。

ドメインの登録方法

RNS Manager を使用してドメインを登録することが可能です。必要なものは：

  • ブラウザ・ウォレット。 Nifty wallet を使用することをお勧めします。
  • RSK ネットワーク・トランザクション費用を支払うための R-BTC、そして
  • RNS ドメイン・サービスに支払うための RIF Tokensです。

これで準備完了です。ドメインを登録しましょう！

  • RNS Managerを参照し、あなたの希望するドメインを検索してください：
  • そのドメインが利用可能であれば、登録（register）をクリックしてください！もし利用可能でなければ、別のドメインを探してください。
  • ドメイン登録する年数を選択し、そのドメインをリクエストしてください。あなたのウォレット内でトランザクションにサインしてください。
  • ドメインが依頼されるまでお待ちください。これには１～２分かかる場合があります。
  • ドメイン登録を完了し、登録（register）をクリックし、最後に要求されるトランザクションにサインしてください。
  • これで完了です！これで RNS ドメインを所有できました！

RSK Testnet ネットワークを使用し、RNS サービスを試すことが可能です。そうするためには、あなたのブラウザ・ウォレットを RSK Testnet に接続してください（Nifty 内でネットワークを切り替えるか、 Metamask 内でその方法をお読みください)。次に、Faucet 内にファンドを取得してください：

  • テストネットR-bTCを取得するための RSK Faucet
  • テストネットRIF Tokenを取得するための RIF Faucet

RNS 利用の利点

  • 柔軟: 必要な回数だけあなたのドメインを更新可能です。あなたのウォレット・アドレスを変更する際は、同じドメインを使用し続けください – ンのアドレス変更方法は？？
  • 分散型: ドメインを管理するための許可を得る必要はありません。一度取得したら、更新のためにはブロックチェーンとのみ対話します – 詳細は、 サービスについてをお読みください
  • 利用が簡単: 誰にでもあなたへの送金を可能にし、長く間違いやすいアドレスの使用を回避します
  • 安価: 現在のドメイン登録費は、1 ドル未満です。ドメインを一度取得したら、必要なだけサブドメインを作成できます – サブドメインの作成方法は？
  • 携帯可能: RNS を Dapp やウォレットで使用してできます。当社のライブラリを利用して、サービスを統合してください – 統合ガイドラインをお読みください

RNS のプラン

当社は現在、プラットフォームの全体的な再設計に取り組んでいます。当社の最新版を試すには、管理者ページ下方にある「try the new admin!」をクリックしてください。

当社は、その他にもたくさんの素晴らしいプロジェクトに取り組んでいます。当社は、Dapp およびウォレットの統合をより速くより効果的にするために、当社のJavaScript ライブラリを積極的に開発しています。オープンソースの rns javascript ライブラリをお試しください！!
