RIFラボがRIF OSを発表

2018年11月18日

RIF Labs はビットコインベースのRSK ネットワークに加え、RIF OS を導入して価値のインターネット（Internet of Value）の実現に向けての一歩を踏み出します

ビットコインは経済的な自由を個人に供与し、そして、私たちのビジョンでは、価値の移転にあたって新たなインターネットを構築する土台を作るべく誕生しました。RIF OS およびRSK、Smart Contract Network は構築されることのみならず、世界中に繁栄と平等をもたらすような一つのビジョンの実現を目指しています 。 このビジョンを成就する上での主たる障壁の一つがコアのプロトコルレベルのテクノロジーと依然新興のブロックチェーンのエコシステムを掌握しようとする開発者との大きなギャップなのです。経験豊富な開発者でさえ、分散型アプリケーション（「dApps」）を創造するために厄介な学習プロセスを経る必要があり、すでにこうしたテクノロジーに通じている開発者は、複数のトークンやプロトコルの統合を伴う長期に及ぶ開発期間と複雑性に煩わされる羽目に陥ることがあります。 本日発表されたRIF OS（Root Infrastructure Framework Open Standard）はこういったギャップの解消を目指します。そして、フルスタックのソリューションであり、分散型のクロスネットワークのブロックチェーン・アプリケーションの展開をいっぞう容易、迅速およびスケーラブルにし、価値のインターネット（Internet of Value）の実現に向けて一歩近づけるのです。 RIF OS 発表の3つの本質的な構成要素にはRIF Labs とRSK Labs の統合、RIF OS プロトコルの公表、RIF ディレクトリ・プロトコルの初の実装の実施が含まれます（RIF OS の名称サービス・コンポーネント）。 RIF およびRSK は共同で安全で柔軟かつ統合的な開発環境を提供します RIF Labs はRSK Labs の買収契約に締結しました。この買収に伴い、RIF Labs はRSK Smart Contract Network とRIF OS プロトコルの両方の開発を主導しつつ、コミュニティの支援の推進を図ります。

RSK は、1) ビットコインが現在のみならず今後も、最も安全で幅広く導入されるネットワークおよび通貨である可能性が極めて高い、2) イーサリアムはアプリケーションと開発者の規模の面で最も成熟してアクティブなスマート・コントラクトのエコシステムを創造した、ならびに、3) RSK は双方の世界のエッセンスを結合したという根拠に立脚しています。結果、イーサリアム対応の仮想マシンが最も安全なネットワークによって保証されるまでとなったのです。 こうした戦略を続行しつつ、RIF OS サービスはRSK を熟知しつつ展開されてきました（つまり、Rootstock）。RIF OS プロトコルは決済からストレージに至る多様な分散型p2p インフラ・サービスへのアクセスをプロジェクトに供与し、市場展開までのスピードを劇的に加速化することで価値を付加し、エコシステム全体の機能性を拡張します。

RSK に関する詳細確認先：

https://www.rsk.co/ https://docs.rsk.rifos.org/

RIF OS プロトコル

RSK（Rootstock）を基盤に展開されるRIF OS プロトコルは決済からストレージに至る多様な分散型p2p インフラ・サービスへのアクセスをプロジェクトに供与し、市場展開までのスピードを劇的に加速化することで価値を付加し、エコシステム全体の機能性を拡張します。 RIF OS プロトコルはエンジニアがブロックチェーン・インフラストラクチャの使用を簡素化する使い勝手に優れたインターフェースと言語ライブラリを通じてより良いdApps をより早く構築するのを可能にします。

こうしたネットワークサービスの決済はRIF トークンにて実施されますが、結果、開発者はあらゆるサービスをRIF OS にシームレスに消費することができるのです

。

発表時、次のプロトコルが指定されます：

    RIF ディレクトリ：仮想通貨とトークン・アカウント対象の読み取りが容易なエイリアスは複雑なブロックチェーンのアドレス共有を伴いません。

    RIF 決済：高速、低コスト、オフチェーンの決済処理ネットワークを実現します。

    RIF データ・フィード：サードパーティのoracling（オラクリング）プロバイダを有効化します

  • RIF ストレージ：安全で分散型およびトラストレスのストレージ・ネットワークへのアクセス。
  • RIF コミュニケーション・チャンネル：安全なコミュニケーション・チャンネルを保証および構築する標準。
開発者とビジネスにとってのブロックチェーン・インフラストラクチャへのアクセスを推進することで、RIF OS プロトコルは、結果、利用者やコミュニティにとってのアクセスを簡素化し、大量導入へのパスを実現します。

RIF ディレクトリとRIF 名称サービス

RIF ディレクトリ・プロトコルを用いた初のサービスが間もなく発表となりますが、RIF 名称サービス（または「RNS」）と呼ばれます。 私たちは、分散型エコシステムが一般の人たちに極めて有益となり、アクセスが簡素化される必要があると確信しています。より包含的なシステム（例：マイクロペイメント等）を作り出す点においては、日常的なユーザーが従来的なネットワークのアドレスを把握および共有するとは考えられません。結局、長ったらしい16進数のウォレットのアドレスをコピー& ペーストするのは厄介で、書き記すのは間違いが伴いやすいことは言うまでもありません。ドメイン名称システムが厄介なインターネットのプロトコルの数字を覚えやすい名称に置き換えたのと同様に、ブロックチェーンベースのサービスが言いやすい便利な名称となるのは必然なのです。すなわち、0x656e72792073726169676d616e とmaryswallet.rsk の違いです。

まさにこれが、RNS が提供することなのです：RIF OS が実現する多様なサービスを他の人たちに推奨する手軽な方法です。これはフォールト・トレランス（fault tolerance）および分散化の付加的な恩恵で、強要する登録管理者や名称サービス担当マネージャーは不要です。

つまり、仮想通貨の加速化と成熟は、1) 開発者が強力なブロックチェーン・アプリケーションを瞬時に構築するワンストップのソースを保有する、ならびに、2) そうしたアプリケーションは幅広い導入との間にわずかなプロセスしか伴わないことを徹底することに大いに左右されます。RIF OS はこの答えを提供し、貴社や貴殿とともに未来を創造してゆければと願っています。
