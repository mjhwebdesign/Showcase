RIF Name Service (RNS) y MyCrypto: RNS Manager 1.0.2 ahora es compatible con MyCrypto

1 de mayo de 2019

En IOVLabs, nos complace anunciar que la nueva versión de RNS Manager les permite a los usuarios conectarse con MyCrypto para cualquier operación, como registrar y administrar nombres.

MyCrypto es una de las herramientas líderes de cliente y de código abierto para generar wallets, manejar tokens RRC-20 e interactuar con la blockchain más fácilmente. Para registrar su dominio en MyCrypto, compruebe el estado del dominio y siga las instrucciones. Durante el proceso de registro y cada acción de nombre, usted verá la opción de salida a MyCrypto para firmar las interacciones de contratos de RNS.

Esta nueva integración muestra una vez más la estrecha relación entre RNS y MyCrypto, ¡la primera wallet integrada compatible con RIF Name Service en usar direcciones “.rsk”!

Los objetivos principales de los servicios RIF consisten en poner la tecnología blockchain al alcance de todos para facilitar su uso. RIF Name Service está diseñado especialmente para que la experiencia del usuario sea más amigable. Esta integración con MyCrypto es un hito importante en esa dirección.
