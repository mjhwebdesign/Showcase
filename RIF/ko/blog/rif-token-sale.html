RIF Token 판매

이 글은 RIF Token 발행 과정의 일부 측면이 화이트페이퍼에 설명되어 있음에도 불구하고, 커뮤니티 내에 여전히 이에 대한 약간의 혼란이 있는 것 같아 이해에 도움을 드리고자 작성한 내용입니다.
2019 년 1 월 6 일

Token 생성 컨트랙트 배치 및 배포

RIF Token 생성 스마트 컨트랙트는 2018년 11월 9일부로 (TGE 날짜) RSK 네트워크에 배치되었습니다. RIF Labs 화이트페이퍼의 토큰 판매 부문
RIF Labs Whitepaper
에서 설명한 대로 10억 개의 RIF Token이 생성되고 할당되었습니다.

컨트랙트 주소

RKS 네트워크 내 RIF Token 생성 스마트 컨트랙트의 주소는 다음과 같습니다:
0x2acc95758f8b5f583470ba265eb685a8f45fc9d5

이 컨트랙트의 활동 및 세부 사항은 RSK 블록체인 탐색기에서 찾아볼 수 있습니다: http://explorer.rsk.co/address/0x2acc95758f8b5f583470ba265eb685a8f45fc9d5

이 컨트랙트는 화이트페이퍼 내 정의된 규칙이 준수됨을 보장하기 위해 일련의 스마트 컨트랙트와 함께 배치되었습니다.

Token 판매 과정

RIF Labs는 비공개 세일 과정에서 구매 가능한 RIF Token을 비공개 기여자들에게 직접 독점적으로 판매하였습니다. RIF Labs는 비공개 세일에 참여한 기여자들을 승인하고, RIF Labs가 고용한, KYC/AML 절차 수행을 전문으로 하는 제3자에게 정보를 제출하는 것을 포함하는 통합적인 KYC 과정을 필수로 거치게 하였습니다. WP와 ECA에는 Token 획득에 있어 기여자들의 목적과 목표, 그리고 그에 따른 사용 가능성에 대한 여러 개의 참조 사항이 포함되어 있습니다.

Token 상환

비공개 세일 중에 판매된 RIF Token 모두는 TGE 날짜에 RIF Labs가 제공한 토큰입니다. 이 RIF Token들은 비공개 세일 기여자들이 TGE 날짜에서 1년이라는 기간 동안 (상환 기간) RIF Labs가 배치한 스마트 컨트랙트에서 상환할 수 있게 제공될 예정입니다. 비공개 세일 기여자들은 RIF Token을 받기 위해 RIF Labs가 보낸 상환 지침에 따라 알맞은 스마트 컨트랙트와 상호 작용을 해야 합니다. 이는 비공개 세일 기여자들이 상환 기간 동안 본인의 토큰을 받을 수 있는 유일한 방법입니다. 상환 기간이 만료되면 스마트 컨트랙트는 기여자가 상환하지 않은 RIF Token 모두를 자동으로 RIF Labs에게 돌려 보내게 됩니다.

보너스 Token 혜택

비공개 세일 중 판매된 RIF Token에는 락업(고정) 규정이 따르지 않습니다. 다만 RIF Labs는 비공개 세일 기여자가 본인의 초기 RIF Token 할당량을 본인의 기존 월렛(지갑)에 각 3개월, 6개월, 그리고 9개월 동안 지속적으로 보관하고 보유하고 있는 경우를 위한 보너스 Token 혜택을 제시하였습니다.
