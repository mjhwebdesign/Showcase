RIF Directory

O RIF Directory, ou diretório, é a camada de identidade e reputação dentro do ecossistema RIF, tendo RNS como um de seus componentes fundamentais.

Ele proporciona usuários e nós com APIs e bibliotecas unificadas para fins de interação com todos os principais protocolos de identidades soberanas.

O RIF Directory visa permitir que usuários controlem facilmente suas IDs para interagirem em economias descentralizadas enquanto constroem uma reputação soberana que possibilitará aos usuários, especialmente aqueles excluídos do sistema financeiro tradicional, participar da economia digital descentralizada do futuro.

O principal objetivo do RIF Directory é proteger os dados pessoais dos usuários, capacitando-os a gerenciar quem pode acessar esses dados e proporcionando controle integral de sua reputação, de modo que possam usá-la para interagir com diversos mercados e plataformas com liberdade para se deslocarem entre espaços diferentes sem perder seu histórico, contatos e valor social.

Especificações do Directory

Definição de RNS

O RIF Name Service (RNS) é um serviço descentralizado que permite que os usuários tenham um domínio legível em qualquer blockchain. O RNS pode ser usado para identificar outros recursos pessoais, como endereços de pagamento, ID, armazenamento ou comunicação.
